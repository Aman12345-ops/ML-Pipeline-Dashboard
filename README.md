# ML-Pipeline-Dashboard
An end-to-end supervised learning project using Linear Regression to predict student performance based on academic and behavioral features.
# Student Performance Prediction 🎓

This project uses Machine Learning to predict students' final scores based on:

- Study Hours
- Attendance
- Subject Marks (Math, Science, English)

## 🚀 Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Plotly / Seaborn

## 📊 Model
- Linear Regression

## 📈 Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run pipeline.py
